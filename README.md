### Core Project: Sentinel Intelligence

I designed and implemented Sentinel Intelligence, a browser-local, privacy-first execution framework and intelligence layer for sensitive agentic workflows.

* **Research Paper Preprint:** Published on Zenodo — [DOI: 10.5281/zenodo.20529516](https://doi.org/10.5281/zenodo.20529516) [![Zenodo DOI Record](https://zenodo.org/badge/DOI/10.5281/zenodo.20529516.svg)](https://doi.org/10.5281/zenodo.20529516)
* **Contributions:** Developed the CAMP (Cumulative Agentic Masking and Pruning) middleware, optimized browser-local WebGPU/WASM LLM inference pipelines, and designed a real-time Cumulative PII Exposure (CPE) state tracking registry to prevent re-identification.
* **Technology Focus:** Next.js, React, TypeScript, WebGPU, WebAssembly (WASM), WebRTC, SQLite (OPFS), WebCrypto API, Vitest.
